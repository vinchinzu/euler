# Project Euler Solutions

## Overview

Solutions to Project Euler problems in multiple languages:
- **Python** - Primary language for new solutions  
- **Ruby** - Legacy solutions (problems 1-110)
- **Rust, C++, C, Fortran** - Additional implementations


## Quick Start

```bash
# Run a Python solution
python python/001.py

# Run with validation  
python validate.py
```

## Solution Standards

- Output only the numeric answer on stdout
- Type hints, ≤88 char lines
- Standalone scripts (no network calls)
- Exit with status 0 on success

## Validation Status

Run `python validate.py` to check all solutions against known answers in `solutions_b.txt`.

Results are saved to `validation_results.json`.

See `agent/AGENTS.md` for detailed workflow and conventions.
